Overview
This assignment confirms your development environment is fully configured. You will verify all required tools are installed and working, create the course folder structure, write and run your first Python script, and document everything in a README. There is no programming challenge yet — the goal is a working environment and the habit of documenting your work.
Requirements
1
Verify all six tools from the zsh terminal

In the VS Code integrated terminal (which must be running zsh), run all six commands and confirm expected output:

echo $SHELL          # Expected: /bin/zsh
zsh --version        # Expected: zsh 5.x or higher
python3 --version    # Expected: Python 3.10 or higher
git --version        # Expected: git version 2.x
pwd                  # Expected: /Users/yourname or /home/yourname
ls                   # Expected: Desktop  Documents  Downloads  etc.

Take a screenshot showing all six commands and their output in a single terminal window.

2
Create the course folder structure

Run the following commands to create the full course folder:

cd ~
mkdir ism3232 && cd ism3232
mkdir module01_setup module02_zsh module03_git_github \
      module04_programming module05_functions \
      module06_oop module07_final_project \
      data screenshots
ls
code .

Take a screenshot of VS Code with the ism3232/ folder open in the Explorer panel, showing all module subfolders in the sidebar.

3
Write and run hello_ism3232.py

Navigate into module01_setup/, create the file, and type the script — do not copy-paste:

hello_ism3232.py
# hello_ism3232.py
# ISM3232 - Business Application Development
# Author: [Your Name]

print('Hello, ISM3232!')

course_name  = 'Business Application Development'
credit_hours = 3
weekly_hours = credit_hours * 2

print(f'Course: {course_name}')
print(f'Expected weekly hours: {weekly_hours}')
print('Environment verified. Week 1 complete.')

Run with python3 hello_ism3232.py. Take a screenshot of the terminal showing the output.

4
Create module01_setup/README.md

Create a README documenting your setup. It must include:

• Your name and the date

• A table showing the verified version of each tool (zsh, Python 3, Git, VS Code)

• A brief description of what you did this week (2–3 sentences)

• An AI use statement: “I did not use AI for this assignment.”

🖥TERMINAL BASICS

📁NAVIGATION & FILE OPS

mkdir nameCreate a new directory
touch file.pyCreate an empty file
tree -L 2Visualise 2-level structure

cp src dstCopy — original stays
mv old newMove/rename — original gone
cat file.txtPrint file contents
head -5 fileFirst 5 lines
tail -5 fileLast 5 lines

⚠ rm SAFETY RITUAL — in this order every time

1pwd
2ls
3rm filename
⚠ rm is permanent. No Trash. No undo.
🐍VIRTUAL ENVIRONMENT

One venv per project. Run once at project start.

1python3 -m venv .venv
2source .venv/bin/activate
3pip install pytest ruff
4pip freeze > requirements.txt
5deactivate          # when done
✓ (.venv) in prompt = active
which python3Must show .venv/bin/python3
pip install -r requirements.txtRecreate from frozen list
.gitignore — create before first git add: .venv/ __pycache__/ *.pyc
⚠ NEVER commit .venv/ — it is large and platform-specific.

🚀PRE-SUBMISSION RITUAL — ALL 9 STEPS

Run every step before every push. In this order. Every time.

1pwd
2tree -L 3
3source .venv/bin/activate
4ruff format .
5ruff check .
6pytest
7git status
8git add .
9git commit -m 'descriptive message'
→git push
⚠ ruff errors → fix before continuing
⚠ pytest failures → fix code before committing

GIT ESSENTIALS

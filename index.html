Unit 1 · Weeks 1–4
Developer Foundations
Build a real developer environment — terminal, virtual environments, Git, and the 9-step ritual.
Overview Cheat Sheet
WEEK 01
Developer Mindset & First Setup
Terminal, VS Code, Python 3, zsh. The 9-step developer ritual. Navigate a project without a GUI. Run your first Python script from the command line.
Reading Lecture Lab Slides
WEEK 02
zsh Navigation & File Operations
cd, ls, mkdir, mv, cp, rm, wildcards. Absolute vs. relative paths. Piping and redirection. Navigate any project directory without a GUI.
Reading Lecture Lab Slides
WEEK 03
Virtual Environments & .zshrc
python3 -m venv, pip install, requirements.txt, .gitignore. Customize .zshrc with aliases and a working prompt. Never install packages globally again.
Reading Lecture Lab Slides
WEEK 04
Search Tools, Ritual & Git
grep, find, ripgrep for codebase search. The 9-step ritual codified. git init/add/commit/push. All future work lives on GitHub from here.

Python Foundations

Overview Cheat Sheet
WEEK 05
Variables, Data Types & Operators
int/float/str/bool. Arithmetic, comparison, and logical operators. Type coercion and common bugs. Business formula: revenue = price × qty.

Conditionals, Loops & Dictionaries
if/elif/else decision trees. for and while loops. Dictionaries for key-value data. Customer lookup tables and discount tier logic.

Functions, Modules & pytest

Debugging, AI Literacy & Midterm Review

Unit 3 · Weeks 10–12
Object-Oriented Design
Model real-world business entities as Python classes — composition, inheritance, and clean design.
Overview Cheat Sheet
WEEK 10
OOP I — Classes & Objects
Classes and instances. __init__, self, and attributes. Methods. Model a business entity — a Product, Order, or Customer — as a Python class.
Reading Lecture Lab Slides
WEEK 11
OOP II — Composition & Inheritance
Composition: objects that own other objects. Inheritance: extend a base class without repeating code. Override __str__ and __repr__ for readable output.
Reading Lecture Lab Slides
WEEK 12
OOP III — Applied Practice & Design
Design patterns in practice. A complete OOP mini-project: model a business domain end-to-end. Code review session and capstone schema planning.
Reading Lecture Lab Slides
Unit 4 · Weeks 13–16
Capstone Build
Ship a complete business application — SQLite database, Streamlit interface, and a controlled AI feature.
Overview Rubric
WEEK 13
Capstone Design & SQL Foundations
Entity-Relationship diagrams. CREATE TABLE, INSERT, SELECT. SQLite from Python's sqlite3 module. Design and build your capstone database schema.
Reading Lecture Lab Slides
WEEK 14
Python + SQL Integration
sqlite3 module. Parameterized queries to prevent injection. SELECT with JOINs. CRUD operations wrapped in clean Python functions.
Reading Lecture Lab Slides
WEEK 15
Streamlit Business Interface
Streamlit widgets, layout, and session state. Connect your app to SQLite. Build an interactive business dashboard that runs in a browser.
Reading Lecture Lab Slides
WEEK 16
GenAI Feature & Final Demo
Anthropic API integration. Add a controlled AI feature to your business app. Final demo day — 5-minute live presentations and course retrospective.

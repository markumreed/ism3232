LISTS

ACCUMULATOR PATTERN

amounts = [200, 1500, 800] total = 0 # initialise BEFORE loop flagged = [] for a in amounts: total += a if a > 1000: flagged.append(a) print(total) # → 2500

DICTIONARIES & LIST-OF-DICTS

req = {"id": 101, "amount": 1200, "status": "Pending"} req["amount"] # → 1200 req["status"] = "Approved" # update records = [ # list of dicts = a table {"id": 1, "amount": 500, "status": "Pending"}, {"id": 2, "amount": 1500, "status": "Pending"}, ] for r in records: if r["status"] == "Pending": print(r["id"], r["amount"])
🧩FUNCTIONS
LIMIT = 1000 # module-level constant def calculate_total(price: float, qty: int) -> float: """Return total including 7% tax.""" return price * qty * 1.07 def requires_review(amount: float) -> bool: """Return True if over approval limit.""" return amount > LIMIT def get_tier(amount: float) -> str: """Return routing tier.""" if amount <= 500: return "auto" elif amount <= 2000: return "manager" else: return "director" # Calling: total = calculate_total(450.00, 3) # → 1444.50 print(requires_review(total)) # → True
parameterVariable in the definition: def f(price)
argumentValue at the call: f(450.00)
returnSends value back to caller — NOT print()
scopeLocal variables exist only inside their function
⚠ Methods must return values. Do not print() inside business logic functions.

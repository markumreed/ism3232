🗣 🗣 Why controls

AI output is probabilistic. It can be wrong. It can be confidently wrong. AI output can reveal sensitive information if PII was sent in the prompt. The six controls are not bureaucracy — they are the minimum responsible design for any production AI feature. In a real company, shipping an AI feature without these controls would be a compliance failure.

🗣 🗣 The review gate pattern

The most commonly missing control: the human review gate. Here is what it looks like. Button 1: 'Generate AI Summary' — AI runs, output shown in an st.info box with a disclosure label. Button 2: 'Save this summary' — user confirms, then the record is saved. If you auto-save the AI output without a second confirmation button, Control 2 is violated.

🗣 🗣 No PII

PII means personally identifiable information: names, employee IDs, amounts, dates. Your prompt should describe the general nature of the request — category, description — without including who made it or how much it costs. This is not just a course requirement. Every API call you make to an external AI service could theoretically log the input. You are responsible for what you send.

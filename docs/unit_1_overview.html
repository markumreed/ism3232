Module 2A & 2B — zsh Basics, Navigation, and File Operations
How do I move around my project and manage files like a developer?
Navigation commands
pwdls / ls -la cd / cd .. / cd ~ mkdirtouch cleartree
File operation commands
cpmv rmcat lesshead / tail code .
Student output
Navigate, create, and organize project files from the terminal. Run a small Python script from zsh.
⚠ Safety rule: always run pwd and ls before rm — demonstrate in lecture and require in screenshots

Module 2C & 2D — Virtual Environments, Dependencies, and .zshrc
How do I manage Python packages and customize my shell?
Virtual environment commands
python3 -m venv .venv source .venv/bin/activate pip install pytest ruff pip freeze > requirements.txt deactivate
Required .zshrc aliases
llc pygs gagcmsg gpgl tree2

W4
Module 2E–2H & Module 3 — Search, Power Nav, Submission Workflow, and Git
How do I find things in a codebase and submit work professionally?
Search and power navigation
rg 'def 'rg 'class ' find . -name '*.py' grep -R 'TODO' . fzfzoxide tree -L 3
Required pre-submission ritual

pwd  ·  tree -L 3  ·  source .venv/bin/activate
ruff format .  ·  ruff check .  ·  pytest
git status  ·  git add .  ·  git commit -m 'msg'  ·  git push

Student output

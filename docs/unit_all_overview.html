Unit 1 · Weeks 1–4 · Developer Foundations, zsh, and Professional Workflow
🔧
Unit philosophy
Students learn zsh not as a technical hobby but because professional developers need to navigate projects, run code, manage dependencies, search codebases, and commit work. Every week includes a small Python task so the terminal context is always connected to application development.
Dev environment→ zsh navigation→ venv + .zshrc→ Search + power nav→ Git + submission workflow

Week 9 · Midterm Practical Exam
📋
Coverage · Structure · Weight: 20%
Tests Units 1 and 2: VS Code workflow, zsh basics and file operations, .zshrc aliases, ripgrep / find / fzf / zoxide, virtual environments, pip, Ruff, pytest, Git/GitHub, variables, conditionals, loops, lists, dicts, functions, debugging, and AI use policy.

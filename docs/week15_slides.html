Why Streamlit?

0–15 min  Why Streamlit? Setup

🧩 Streamlit turns Python scripts into browser-based business tools

Streamlit lets you build an interactive web interface for your Python application without writing HTML, CSS, or JavaScript. Every widget — text inputs, buttons, dropdowns, tables — is a Python function call. The interface re-runs the entire script top to bottom whenever the user…
“Because the entire script reruns on every interaction, anything you compute at the top of the script is recomputed every time. This is why database calls go inside tab blocks or conditional blocks — not at the top level.”

🖥 Terminal

# Activate venv first
source .venv/bin/activate
pip install streamlit
pip freeze > requirements.txt

# Run your app
streamlit run app.py

Core Streamlit Components

15–45 min  Build Streamlit app together

🐍 Code

import streamlit as st

# Page setup (call once, at the top)
st.set_page_config(page_title="Request Tracker", layout="wide")

# Text display
st.title("Business Request Tracker")
st.header("Submit a New Request")
st.write("Any text, markdown, or value — very flexible")

# Input widgets — each returns a Python value
requester = st.text_input("Your name")
category  = st.selectbox("Category", ["Travel", "Equipment", "Software"])
amount    = st.number_input("Amount ($)", min_value=0.0, step=10.0)
notes     = st.text_area("Notes (optional)")

# Button — returns True only on the click that triggered the rerun
if st.button("Submit Request"):
    if requester and amount > 0:
        add_record(requester, category, amount, notes)
        st.success(f"Request submitted for {requester}")
    else:
        st.error("Name and amount are required.")

# ⋯ continues

loops and the accumulator pattern
for loop

    total += amount
    if amount > 1000:
        flagged.append(amount)
Accumulator — OUTSIDE the loop
total   = 0.0   # ← before loop
flagged = []    # ← before loop

for amount in amounts:
    total += amount  # updates each time
✗ Accumulator inside the loop — resets every iteration
for amount in amounts:
    total = 0.0     # resets each time!
    total += amount # always = last amount
✓ Initialise before the loop
total = 0.0             # once, before loop
for amount in amounts:
    total += amount # accumulates correctly

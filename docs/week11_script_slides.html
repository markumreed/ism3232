🗣 🗣 has-a vs is-a

Composition is a has-a relationship. A RequestManager HAS A list of requests. Inheritance is an is-a relationship. A TravelRequest IS A BusinessRequest. Use composition when one class manages a collection of another. Use inheritance when one class is a specialised version of another.

🗣 🗣 super().__init__()

super() refers to the parent class — BusinessRequest. super().__init__() calls the parent's __init__ to set up the inherited attributes: req_id, requester, category, amount, and status. Without this call, the child instance would be missing all those attributes and you would get AttributeError the first time you tried to access them.

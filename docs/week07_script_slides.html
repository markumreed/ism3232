🗣 🗣 Constants rule

Module-level constants — those ALL_CAPS names at the top of your file — are visible inside functions. Local variables are not. Use return to bring values out of a function. Do not try to read local variables from outside.

SAY: Module-level constants — those ALL_CAPS names at the top of your file — are visible inside functions. Local variables are not. Use return to bring values out of a function. Do not try to read local variables from outside.

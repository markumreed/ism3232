SQL Foundations

20–45 min  SQL fundamentals — live demo

🧩 SQL is how business data is stored and retrieved at scale

SQLite is the database engine for this course — it stores the entire database in a single .db file with no server required. The SQL you learn here (CREATE TABLE, INSERT, SELECT, WHERE, UPDATE, GROUP BY) applies directly to PostgreSQL, MySQL, and every production database system.…
“Using the domain from your proposal, write your CREATE TABLE statement now. Include all columns you identified in the proposal's Records Stored field. Add appropriate types and constraints.”

🐍 Code

-- Create a table (run once)
CREATE TABLE requests (
    id        INTEGER PRIMARY KEY,
    requester TEXT    NOT NULL,
    category  TEXT    NOT NULL,
    amount    REAL,
    status    TEXT    DEFAULT 'Pending',
    notes     TEXT
);

-- Insert records
INSERT INTO requests (requester, category, amount)
VALUES ('Taylor', 'Travel', 1200.00);

-- Query all records
SELECT * FROM requests;

-- Filter records
SELECT * FROM requests WHERE status = 'Pending';
SELECT * FROM requests WHERE amount > 1000;

-- Sort results
SELECT * FROM requests ORDER BY amount DESC;

Running SQL in the Terminal with sqlite3

45–60 min  sqlite3 shell exploration

🐍 Code

# Open or create a database file
sqlite3 requests.db

-- Inside the sqlite3 shell:
.tables                         -- list all tables
.schema requests                -- show CREATE TABLE statement
.mode column                    -- formatted column output
.headers on                     -- show column names
SELECT * FROM requests;         -- query the table
.quit                           -- exit the shell

🗣 🗣 Why context manager

The with statement is a context manager. When the block exits — whether normally or due to an error — the connection is automatically committed and closed. Without the context manager, you must manually call conn.commit() and conn.close(). Forget either one: data is not saved, or the connection leaks. Use the context manager every time.

🗣 🗣 Why it matters

When you pass records to st.dataframe() in Streamlit, it expects a list of dicts. If you return a list of tuples, Streamlit shows numbered columns, not named ones. dict(row) converts each Row object to a plain Python dict. row_factory must be set BEFORE the execute() call to take effect.
